# NodeJS TS starter

Project starter using node, ts and webpack

## Installation

-   Clone this repo

-   `npm install`

-   Use next scripts:
    1. `build:dev` - build dev version of project
    2. `build:prod` - build production version of project
    3. `start:dev` - start dev version of project in watch mode
    4. `start:prod` - start production version of build
    5. `run:prod` - start watch on builded dev version
